### 🎓 iSchool — Student Management App
Une application web de gestion des étudiants, développée en HTML, CSS et JavaScript . Elle permet d'ajouter, modifier, supprimer, rechercher et exporter les fiches étudiants sous forme de cartes visuelles.

### Description
Cette application permet aux differentes ecoles de gerer facilement et librement leurs etudiants , chaque etudiant sera :
*** Enregistrer sous forme de carte avec leurs nom, prenom, sexe, email, age et photo de profil ***

### ✨Fonctionnalités 
➕ Ajouter un étudiant avec photo de profil
✏️ Modifier les informations d'un étudiant existant
🗑️ Supprimer un étudiant ou tous les étudiants
🔍 Rechercher en temps réel (nom, prénom, matricule, email)
📥 Télécharger la carte d'un étudiant en image .png
📊 Exporter la liste complète vers un fichier Excel .xlsx
